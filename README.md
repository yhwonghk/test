# 中国象棋教学网站

一个专业的中国象棋教学平台，提供从基础规则到高级战术的全方位学习体验。

## 🚀 功能特色

- **🎯 系统教学** - 从基础知识到高级战术的完整学习体系
- **🎮 互动练习** - 实时对弈练习，支持棋子移动和游戏状态管理
- **📚 规则详解** - 详细的象棋规则说明和走法图解
- **🎨 现代UI** - 美观的用户界面，流畅的动画效果
- **📱 响应式设计** - 完美适配桌面和移动设备
- **⚡ 高性能** - 基于React和TypeScript构建，性能优异

## 🛠️ 技术栈

- **前端框架**: React 18 + TypeScript
- **构建工具**: Vite
- **样式框架**: Tailwind CSS
- **动画库**: Framer Motion
- **图标库**: Lucide React
- **路由**: React Router DOM

## 📦 安装和运行

### 环境要求

- Node.js 16.0 或更高版本
- npm 或 yarn

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

访问 [http://localhost:3000](http://localhost:3000) 查看网站

### 构建生产版本

```bash
npm run build
```

### 预览生产版本

```bash
npm run preview
```

## 🎯 项目结构

```
src/
├── components/          # 可复用组件
│   ├── Header.tsx      # 网站头部导航
│   ├── ChessBoard.tsx  # 象棋棋盘组件
│   └── ChessPiece.tsx  # 象棋棋子组件
├── pages/              # 页面组件
│   ├── Home.tsx        # 首页
│   ├── Tutorial.tsx    # 教学页面
│   ├── Practice.tsx    # 练习页面
│   └── Rules.tsx       # 规则页面
├── types/              # TypeScript 类型定义
│   └── chess.ts        # 象棋相关类型
├── utils/              # 工具函数
│   └── chessLogic.ts   # 象棋游戏逻辑

└── index.css           # 全局样式
```

## 🎲 核心功能

### 象棋棋盘
- 完整的10x9象棋棋盘
- 楚河汉界分隔线
- 九宫格标识
- 坐标显示（可选）

### 棋子系统
- 完整的象棋棋子集合（帅/将、仕/士、相/象、马、车、炮、兵/卒）
- 准确的走法规则实现
- 棋子选择和移动动画
- 吃子逻辑

### 游戏逻辑
- 回合制游戏流程
- 移动验证
- 将军检测
- 游戏状态管理
- 移动历史记录
- 悔棋功能

### 教学内容
- **基础知识**: 棋盘介绍、棋子认识
- **走法规则**: 每个棋子的详细走法说明
- **基础战术**: 将军应将、常见杀法、开局原则

## 🎨 设计特色

- **现代化界面**: 使用渐变背景和毛玻璃效果
- **流畅动画**: Framer Motion 提供的丰富动画效果
- **响应式布局**: 完美适配各种屏幕尺寸
- **直观交互**: 清晰的视觉反馈和用户引导

## 📚 学习路径

1. **首页** - 了解网站功能和特色
2. **规则页面** - 学习象棋的基本规则和特殊规则
3. **教学页面** - 系统学习象棋知识
   - 基础知识：棋盘和棋子介绍
   - 走法规则：各棋子的具体走法
   - 基础战术：常用战术和技巧
4. **练习页面** - 实际操作练习
